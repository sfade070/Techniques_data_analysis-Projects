Objective¶

Language classification is a supervised machine learning task of mapping a text to a unique language from a set of trained languages. It involves building language models from a lot of text data of respective languages and then identifying the test text data among the trained language models. In this project we try to build a language classifier for identifying four languages: English(en), German(de), Spanish(es) and French(fr).

We build our language models based on letter bi-grams from training text data for four languages. A bi-gram is a 2-character slice of a word. We also consider spaces as a character of the bi-grams in order to mark the distinction between beginning and end of word and the inner bi-grams. For example, a sentence she sings will have the following bi-grams:

s, sh, he, e, s, si, in, ng, gs, s

In a bi-grams model, we tokenize the text in characters (including spaces or underscores), generate bi-grams from these tokens and count the occurrence of each bi-grams. We then make a sorted list of bi-grams in descending order of their frequencies. The most frequent ones form a language model for training corpus of a particular language. We repeat this process to train a language model for each language of interest, here we have four.

For testing the model, we select large number (5000) of text sentences from a particular language (with known language label). We then tokenize each text sentence to bi-grams and their frequencies, and compare bi-grams to the bi-grams of each of 4 trained models and sum up the frequencies. We select the language with maximal sum as the language of a test sentence. We repeat this process 5000 times for each sentence in a test corpus of each language. Then we create a confusion matrix to check the accuracy of predicting the language of each test corpus.

To summarize, in the rest of this project, we walk through the following steps (including text processing whenever needed) and demonstrate the result and accuracy of our language model:

  • Download the text corpus for 4 selected languages (English, German, Spanish and French) from the Europal corpus available online at http://www.statmt.org/europarl/. The text corpus are downloaded into 4 folders labeled as en, de, es and fr. Split the text corpus of each folder to train and test folders.
  • Build bi-grams model for 4 selected languages (English, German, Spanish and French). The training corpus consists of all texts in the train folder.
  • Test the language model on the test corpus consists of 5000 sentences for each language taken from the test folder.
  • Test the language model on some random texts (sentence or word).

Build Bi-Grams Language Models¶

This section include the following steps all done in the code below:

  • Preprocessing of text in the training corpus which can improve the accuracy of the language identification task. We have considered the following preprocessing steps before creating bi-gram language model:

    • Texts have been extracted from each text file which is originally in lxml format.
    • Texts were converted to lower case.
    • The digits were removed from the texts.
    • Punctuation marks and special characters were removed.
    • All the texts in the train folder for each language were concatenated with space in between.
    • Series of contiguous white spaces were replaced by single space.

  • Extracting the bi-grams and building language models:

    • Finding bi-grams and their frequencies will be achieved through BigramCollocationFinder function in nltk in Python.
    • Using other functions of nltk to find out list of bi-grams sorted with number of occurrences for each language. Keep only bi-grams with frequencies equal or higher than 5.
    • Saving a bi-grams language model for each of the 4 languages in npy format.

import numpy as np
import pandas as pd
import os 
import glob
import urllib
from bs4 import BeautifulSoup
from nltk.collocations import BigramCollocationFinder
import re
import codecs
import string

def train_language(label):
    text_all = []
    translate_table = dict((ord(char), ' ' ) for char in string.punctuation)
    filenames = os.listdir('train/' + label + '/')
    path = os.getcwd() + '/train/' + label + '/'
    for file in filenames:
        with codecs.open(path + file ,'r', encoding='utf-8') as infile:
            txt = infile.read()
        soup = BeautifulSoup(txt,'lxml')
        text = soup.get_text()
        text = text.replace('\n','')
        text = ' '.join(text.split()[0:])
        text = text.lower()   # to lower case
        text = re.sub(r'\d+', '', text) # remove digits
        text = text.translate(translate_table) # remove punctuations
        
        if len(text) != 0:
            text_all += text
            text_all.append(' ') # append texts with space
            
    all_str = ''.join(text_all)
    all_str = re.sub(' +',' ',all_str) # replace series of spaces with single space
    seq_all = [i for i in all_str] 
        
    # extracting letter bi-grams and sorting them according to their frequencies
    finder = BigramCollocationFinder.from_words(seq_all)
    finder.apply_freq_filter(5) # keep bigrams with freuencies equal or greater than 5
    bigram_model = finder.ngram_fd.items()
    bigram_model = sorted(finder.ngram_fd.items(), key=lambda item: item[1],reverse=True) # sort based on frequency
    np.save(label+".npy",bigram_model) # save language model
        
if __name__ == "__main__":
    labels = ['en', 'de', 'es', 'fr']
    for label in labels:
        train_language(label)

Evaluation on Test Corpus¶

Test corpus contains 5000 sentences per language. To classify a text sentence among the language models, we calculate the distance of the input sentence with the bi-gram language model. Then we choose the language with the minimal distance as the language of the input sentence.

To carry with this procedure first, we preprocess the input sentence and extract the bi-grams from the input sentence. Then, we calculate the frequencies of each of these bi-grams from the language model and summed them up. Note that in order to remove any bias due to size of the training text corpus of each language, it is necessary to normalize the summed up frequency for each language by sum of frequencies of all the bi-grams in the respective language. Moreover, we multiply the frequencies by a factor of 5000 to avoid the case when normalized frequency become zero. The following equation is used for calculating the normalized frequencies: $$ f_n(j)=\frac{5000\sum_{j=1}^{k}f(i,j)}{\sum_{j=1}^{m}f(i,j)} $$

where $f_n(j)$ is the normalized frequency sum of language, $f(i,j)$ is the frequency of the $i^{th}$ bi-gram in $j^{th}$ language, $k$ is the number of bi-grams which occur in the test sentence and $m$ is the total number of bi-grams in the respective language model.

def test_language(total):
    
    lang_name = ['en', 'de', 'es', 'fr']
    model = [np.load(lang+".npy", allow_pickle=True) for lang in lang_name]
    df_confusion = pd.DataFrame()
    
    for i, sent in enumerate(sents_sample):
        enp = 0
        dep = 0
        esp = 0
        frp = 0
        ccd = 0
        for j in range(0,len(sent[0])-1):
            finder = BigramCollocationFinder.from_words(sent[0][j])
            freq_sum = np.zeros(4)
            for k,v in finder.ngram_fd.items():
                isthere = 0
                for i,lang in enumerate(lang_name):
                    for key,f in model[i]:
                        if k == key:
                            freq_sum[i] = freq_sum[i]+(f*5000)/total[i]
                            isthere = 1
                            break
                if isthere == 0:
                    freq_sum[i] = freq_sum[i] + 1
 
            max_val = freq_sum.max()
            index= freq_sum.argmax()
        
            if max_val != 0:
                if lang_name[index] == lang_name[0]:
                    enp = enp + 1
                elif lang_name[index] == lang_name[1]:
                    dep = dep + 1
                elif lang_name[index] == lang_name[2]:
                    esp = esp + 1
                elif lang_name[index] == lang_name[3]:
                    frp = frp + 1
            else:
                ccd = ccd + 1
    
        df_confusion = df_confusion.append([[enp, dep, esp, frp, ccd]],ignore_index=True)
        
    return df_confusion
